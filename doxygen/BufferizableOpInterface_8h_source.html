285 /// BufferizationState provides a variety of helper functions for dealing with

287 /// `BufferizableOpInterface::bufferize` implementation should utilize the
288 /// following helper functions.
289 ///
290 /// * `createAlloc` / `createDealloc` / `createAllocDeallocPair` creates ops

293 /// * `getResultBuffer` returns the memref buffer for a given tensor OpResult.

330  /// Starting from `value`, follow the use-def chain in reverse, always
331  /// selecting the aliasing OpOperands. Find and return Values for which
332  /// `condition` evaluates to true. OpOperands of such matching Values are not
333  /// traversed any further.
334  ///
335  /// When reaching the end of a chain (BlockArgument or Value without aliasing
336  /// OpOperands), also return the last Value of that chain.
337  ///
338  /// Example:
339  ///
340  /// 8

346  /// +----------+----------+----------+
347  /// |

349  ///

351  /// starting the traversal from Value 1, the resulting SetVector is:

527  LogicalResult bufferize(Operation *op, OpBuilder &b,

529  auto isaTensor = [](Type t) { return t.isa<TensorType>(); };

531  any_of(op->getResultTypes(), isaTensor))
